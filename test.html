Better Text Translator Test Page

How to Test

  1. Load this extension in Chrome by going to chrome://extensions/
  2. Enable Developer mode and click "Load unpacked"
  3. Select the extension directory
  4. Configure your API settings by clicking the extension icon
  5. Come back to this page and double-click on any text below

Test Words and Phrases

Double-click on any of these words or phrases to test translation:

  • like - The example word you mentioned
  • hello - Simple greeting
  • beautiful - Adjective
  • running - Verb
  • friendship - Abstract noun
  • computer - Technical term
  • delicious - Descriptive word

Phrases to Test

  • good morning - Common greeting
  • how are you - Question
  • thank you - Gratitude expression
  • I love coffee - Simple sentence
  • see you later - Farewell phrase

Expected Behavior

When you double-click on text OR use modifier+click, you should see:

  1. A popup appearing in the top-right corner
  2. The original text displayed
  3. The translation in your target language (default: Chinese)
  4. NEW: Two example sentences using the word/phrase with their translations

Note: Modifier+click can select individual words automatically, even if no text was pre-selected!

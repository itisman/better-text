Base URL Configuration Test

Test Instructions:

  1. Open the extension options page (click extension icon or go to chrome://extensions and click "Options")
  2. Select an API provider (DeepSeek or OpenAI)
  3. Notice the new "API Base URL" field appears below the API Key field
  4. The placeholder shows the default URL for each provider:
    • DeepSeek: https://api.deepseek.com (default)
    • OpenAI: https://api.openai.com (default)
  5. You can leave it empty to use the default URL
  6. Or enter a custom base URL like https://proxy.example.com
  7. Save settings and test the connection

Expected Behavior:

  • ✅ Base URL field should show/hide based on provider selection
